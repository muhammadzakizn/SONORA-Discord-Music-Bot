# Bot Changelog v3.16.0 - YTDLP API Integration

**Release Date:** 2025-12-19

## New Features

### YTDLP API Integration
Bot sekarang menggunakan YTDLP API (`localhost:9072`) untuk semua operasi audio.

**Flow:**
1. Try API first (`/api/ytdlp/...`)
2. Fallback ke direct CLI jika API tidak available

**Methods yang diupdate:**
- `search()` - API search → CLI fallback
- `get_stream_url()` - API stream-url → CLI fallback
- `_download_from_ytdlp()` - API download → CLI fallback

### Files Created
- `services/audio/ytdlp_client.py` - HTTP client untuk YTDLP API

### Files Modified
- `services/audio/youtube.py` - Use API client dengan CLI fallback

### Benefits
- Centralized yt-dlp operations melalui API
- API sudah handle cookies dan 6-second pre-buffering
- Fallback otomatis ke CLI jika API tidak available
