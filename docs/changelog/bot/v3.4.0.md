# Bot API Changelog - v3.4.0

## 2024-12-12

### Added
- **Queue Remove Endpoint** - `POST /api/queue/{guild_id}/remove/{position}`
- **Queue Move Endpoint** - `POST /api/queue/{guild_id}/move`
- **Discord Channel Notifications** - Control actions send embed notifications

### Changed
- Control API accepts `username` in request body for attribution
