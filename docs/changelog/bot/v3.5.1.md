# Bot Changelog v3.5.1 - MFA Persistence

**Release Date:** 2025-12-14

## Changes

### Auth System
- TOTP secrets now properly stored encrypted in database via `/api/auth/mfa/totp/setup`
- Login verification endpoint `/api/auth/mfa/totp/verify` retrieves stored secret from DB
- Bot API serves as single source of truth for all MFA operations
