# SONORA Bot v3.26.0

**Release Date:** 2026-01-03

## AI Customer Support System - Phase 1

### New Features

#### `/support` Command
- Open AI-powered support session via DM
- AI assistant using Google Gemini API
- Intent detection (question, feedback, issue, live support)
- Context-aware responses about SONORA features

#### Support Forms (Discord Modals)
- **Feedback Form** - Submit suggestions/feedback
- **Issue Report Form** - Report bugs with steps to reproduce
- **Live Support Form** - Request to talk to developer

#### Ticket System
- Random ticket ID format: `SNRA-XXXX-XXXX`
- Status tracking: PENDING → IN_PROGRESS → WAITING_USER → RESOLVED → CLOSED
- SQLite database for ticket storage
- Developer notification via DM when new ticket created

### API Endpoints
- `GET /api/support/tickets` - List tickets
- `GET /api/support/tickets/{id}` - Get ticket detail
- `PUT /api/support/tickets/{id}/status` - Update status
- `POST /api/support/tickets/{id}/message` - Send message to user

### Configuration
- Added `GEMINI_API_KEY` environment variable
- Requires: `pip install google-generativeai`
