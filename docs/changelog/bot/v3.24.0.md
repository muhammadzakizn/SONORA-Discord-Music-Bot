# SONORA Bot v3.24.0 - Lyricify API Integration

**Release Date:** 2025-12-22

## New Features

### LyricifyApi C# Microservice
- New C# .NET 9.0 Web API microservice for syllable-timed lyrics
- Wraps Lyricify-Lyrics-Helper library for QQ Music, Netease, Kugou
- Auto-starts with launcher on port 5050

### Python Lyricify Client
- Added `services/lyrics/lyricify_client.py`
- Async client for LyricifyApi with health check
- QRC format parsing for syllable timing

### Config Updates
- `LyricsSource.LYRICIFY` added to constants
- `LYRICIFY_API_PORT = 5050` in launcher

### Launcher Updates
- Auto-starts LyricifyApi microservice in production mode
- Graceful shutdown handling for LyricifyApi
- Warning if .NET SDK not installed
