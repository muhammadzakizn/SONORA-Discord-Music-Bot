# Bot Changelog v3.14.0 - YTDLP Audio API

**Release Date:** 2025-12-19

## New Features

### YTDLP Audio API
REST API terpisah untuk audio streaming via yt-dlp, mencegah masalah cookies/streaming expired.

**New Endpoints:**
- `GET /api/ytdlp/search` - Search lagu dari YouTube Music
- `GET /api/ytdlp/stream-url` - Get fresh stream URL on-demand
- `GET /api/ytdlp/download` - Download audio file
- `GET /api/ytdlp/stream` - Proxy audio streaming
- `GET /api/ytdlp/info` - API status dan info

### Files Added
- `web/api/ytdlp_api.py` - Flask blueprint dengan 5 endpoints

### Files Modified
- `web/api/app.py` - Register YTDLP API blueprint

### Benefits
- Fresh stream URL setiap request, menghindari URL expired
- Cookies dikelola dengan baik di sisi API
- Fallback ke download jika streaming gagal
- Proxy streaming untuk menghindari 403 errors
