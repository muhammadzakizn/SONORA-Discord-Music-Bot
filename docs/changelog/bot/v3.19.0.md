# Bot Changelog v3.19.0 - Synchronized Playback Dashboard

**Release Date:** 2025-12-19

## New Feature

### Synchronized Playback Dashboard
Dashboard sekarang tersinkronisasi dengan Discord playback.

**Flow:**
1. Bot sends `track_preparing` → Dashboard shows loading
2. Bot sends `track_ready` → Dashboard shows track with animation
3. Dashboard sends `ACK` → Bot confirmed
4. Bot sends `playback_started` → Dashboard starts lyrics sync

### Files Created
- `web/api/track_state.py` - TrackStateManager for state sync

### Files Modified
- `web/api/flask_app.py` - Added WebSocket events and API endpoints
- `ui/media_player.py` - Added dashboard notification methods

### API Endpoints
- `GET /api/state/<guild_id>` - Get track state
- `POST /api/state/<guild_id>/ack` - Send ACK

### WebSocket Events
- `track_preparing` - Track is being prepared
- `track_ready` - Track ready to play
- `lyrics_ready` - Lyrics loaded
- `playback_started` - Playback started
- `track_ended` - Playback ended
