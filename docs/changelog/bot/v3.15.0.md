# Bot Changelog v3.15.0 - One Track One Process Mode

**Release Date:** 2025-12-19

## New Features

### One Track One Process Mode
Mode baru untuk playlist/album processing yang lebih sequential dan hemat resources.

**Cara Aktivasi:**
```env
ONE_TRACK_ONE_PROCESS=true
```

**Perilaku saat aktif:**
- Buffer size = 1 (bukan 3)
- Hanya siapkan 1 track ke depan pada satu waktu
- Proses: check stream → yt-dlp → musicdl → play → cleanup
- Lebih hemat memory dan bandwidth

### Files Modified
- `config/settings.py` - Tambah `ONE_TRACK_ONE_PROCESS` setting
- `.env.example` - Dokumentasi setting baru
- `services/audio/playlist_cache.py` - Dynamic `buffer_size` property
- `ui/media_player.py` - Update `_prefetch_next_track` limit

### Default Behavior
- `ONE_TRACK_ONE_PROCESS=false` (default) = buffer 3 tracks ahead
- `ONE_TRACK_ONE_PROCESS=true` = buffer 1 track ahead
