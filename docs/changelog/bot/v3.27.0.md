# SONORA Bot v3.27.0

**Release Date:** 2026-01-03

## AI Support System Improvements

### Changes

#### AI Intelligence Improvements
- Updated system prompt with complete SONORA feature list
- AI now properly answers feature questions instead of redirecting to feedback form
- Improved keyword detection to distinguish between asking about features vs requesting new features
- Added question patterns: "apa saja fiturnya", "bisa apa", "how to", etc.

#### Removed `/support` Command
- Users now simply DM the bot directly for AI support
- No need for slash command - more intuitive experience
- DM handler still active for 24/7 AI assistance

#### Welcome Message Enhancement  
- Added "Need Help?" section in welcome message
- Informs users they can DM bot for AI support
- Bilingual (Indonesian/English) messaging

#### Bug Fixes
- Fixed messages not appearing in Admin Support Dashboard
- `get_all_tickets` now properly loads messages for each ticket
- Fixed Discord DM not being sent when developer replies
- Corrected bot import path in support_api.py (uses app.py instead of flask_app.py)
- Fixed Support API registration (now registered in app.py used by main.py)
