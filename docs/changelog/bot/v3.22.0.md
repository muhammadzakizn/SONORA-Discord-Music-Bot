# Bot Changelog v3.22.0 - Open Dashboard Access

**Release Date:** 2025-12-19

## New Features

### Voice Channel Verification
- Control endpoints now verify user is in same voice channel as bot
- Returns 403 error with voice channel name if not in voice
- Applies to: pause, resume, skip, stop, queue remove

### User Attribution in Notifications
- Discord notification embeds now show `@user` mention
- Format: `[Action] via Admin Dashboard by @username`
- Helps track who made changes from dashboard

### Server List Separation
- Dashboard now shows all servers user is in
- Separated into two sections:
  - **Servers You Manage** (Owner/Admin permissions)
  - **Other Servers** (Member - join voice to control)

## Files Modified
- `web/api/app.py` - Voice check + @user_id in control/queue endpoints
- `web/src/app/admin/guilds/[guildId]/page.tsx` - Removed isManaged restrictions
- `web/src/app/admin/guilds/page.tsx` - Two-section display
