# SONORA Bot v3.28.0

**Release Date:** 2026-01-03

## Major Update: Server Migration & Improvements

### Infrastructure Changes
- **Server Migrated** from Pterodactyl (Linux) to Windows Server
- Improved stability and performance
- All services fully operational

### AI Support System
- **Groq Integration** - Free AI with 14,400 requests/day limit
- Fallback chain: Groq → DeepSeek → Gemini
- Faster response times

### Voice Channel Protection
- **Playback controls now require voice presence**
- Users must be in same voice channel as bot to:
  - Pause/Resume/Skip/Stop
  - Change volume
  - Manage queue (clear/shuffle)
- Error message guides users to correct channel

### Language & UI Cleanup
- All bot messages now in **English only**
- Reduced emoji usage for cleaner messages
- Menu controls simplified
- Admin dashboard labels updated to English

### Admin Dashboard Changes
- Status controls **removed from admin dashboard**
- Only developer dashboard can change ticket status
- Improved ticket management workflow

### Files Modified
- `commands/control.py` - Voice channel check for commands
- `ui/menu_view.py` - Voice check + English messages
- `services/support/support_ai.py` - Groq integration
- `commands/support.py` - English messages
- `commands/info.py` - English labels
- `web/src/app/admin/support/page.tsx` - Status controls removed
