# Web Changelog - v3.29.0

## December 25, 2025

### Immersive Fullscreen Lyrics Experience

Clean fullscreen lyrics mode that hides all UI chrome for an immersive experience.

#### Features
- NavLiquidGlass, scroll-to-top button hidden during fullscreen lyrics
- Admin Dashboard header, sidebar, notifications, profile hidden
- Footer hidden during fullscreen mode
- Content area expands to full screen without padding
- Swipe up animation when opening fullscreen lyrics
- Swipe down animation when closing
- Empty state displayed when no track is playing or track ends

#### Architecture
- New `FullscreenLyricsContext` for shared state management
- Context provider integrated at app root level
- Components respond to `isFullscreenLyricsOpen` state
- Spring-based animations for smooth open/close transitions
