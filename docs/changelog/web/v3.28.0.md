# Web Changelog - v3.28.0

## December 23, 2025

### Top 10 Tracks Feature

New "Top 10 Tracks" section on Admin Dashboard with monthly statistics.

#### New Components
- **TopTracksCarousel**: Horizontal scrollable carousel showing top 10 tracks
- **Month/Year Selector**: Navigate through monthly and yearly listening history
- **ListeningHistorySection**: History management in Settings page

#### Features
- Display top 10 tracks with album artwork and rank badges
- Monthly and yearly play statistics
- Play count and total duration display
- Shimmer loading placeholders
- Delete history by track, month, or year

#### Privacy & Data
- Added listening history disclosure to Privacy Policy (section 5.1)
- Annual auto-cleanup on January 1st
- User control to delete data anytime from Settings

#### API Endpoints
- `GET /api/top-tracks/monthly` - Monthly top tracks
- `GET /api/top-tracks/yearly` - Yearly top tracks
- `GET /api/history/months` - Available history months
- `GET /api/history/summary` - User history summary
- `DELETE /api/history/delete-month` - Delete monthly history
- `DELETE /api/history/delete-year` - Delete yearly history
- `GET /api/artwork/search` - Search album artwork

#### Database
- Added `artwork_url` column to `play_history` table
- New aggregation methods for top tracks statistics
