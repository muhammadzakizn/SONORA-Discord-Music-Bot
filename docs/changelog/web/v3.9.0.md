# Web Changelog v3.9.0 - Apple Music Style Lyrics

**Release Date:** 2025-12-18

## New Features

### Fullscreen Lyrics Player
- New Apple Music-inspired fullscreen lyrics display
- Per-word animated highlighting synchronized with playback
- Dynamic blur background from album artwork
- Vinyl disc animation when playing

### Lyrics Controls
- Play/Pause/Skip/Stop controls within lyrics view
- Collapsible queue panel
- Romanization toggle for Asian lyrics (Japanese, Korean, Chinese)
- Progress bar with current time display

### Backend API
- New `/api/guild/<guild_id>/lyrics` endpoint
- Returns synced lyrics with word-level timing
- Real-time playback position for smooth sync

## UI Improvements

### Guild Detail Page
- Added "Lyrics" button (Mic icon) in playback controls
- Opens fullscreen lyrics player when playing music

### CSS Animations
- Added slow spin animation for vinyl effect
- Added scrollbar-hide utility for lyrics container
- Added per-word glow animation
