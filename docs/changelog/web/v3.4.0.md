# Web Dashboard Changelog - v3.4.0

## 2024-12-12

### Added
- **Discord Notifications** - Control actions now send embed messages to Discord channel
- **Queue Management UI** - Move up/down and delete buttons on queue items
- **Smooth Progress Bar** - Uses requestAnimationFrame for fluid animation
- **GIF Image Support** - Server icons and artwork support all image formats

### Changed
- Control API now accepts username for notification attribution
- Progress bar transitions smoothly instead of jumping every 3 seconds

### Fixed
- Broken album artwork now shows fallback placeholder
- Server icon (.gif) display fixed
