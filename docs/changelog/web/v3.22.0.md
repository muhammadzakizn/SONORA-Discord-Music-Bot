# Web Changelog v3.22.0 - Open Dashboard Access

**Release Date:** 2025-12-19

## Changes

### All Users Can Control
- Removed admin-only restrictions on playback controls
- Any user in voice channel can control music
- Backend verifies voice channel presence

### Server List Redesign
- Two-section layout:
  - Servers You Manage (yellow badge)
  - Other Servers (cyan badge, "Join voice to control")

### Voice Channel Error Handling
- Special error message when not in voice
- Shows which voice channel to join
