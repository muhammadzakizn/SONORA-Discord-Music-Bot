# Web Changelog v3.8.4 - MFA & Login Fixes

**Release Date:** 2025-12-18

## Bug Fixes

### MFA Login Flow
- Fixed all MFA methods not working during OAuth flow
- Root cause: MFA functions required session `user` which was null during OAuth
- Changed condition to accept EITHER user OR authUser from OAuth callback
- Fixed: handlePasskeyVerify, sendVerificationCode, handleVerifyCode

### NavLiquidGlass Profile
- Fixed profile not showing after MFA login until manual refresh
- Changed router.push to window.location.href for full page reload
- SessionProvider now re-reads cookies immediately after login

### Servers Page Stats
- Stats now show only user's managed servers (owner/admin)
- Previously showed total bot-wide stats incorrectly
- Total Servers, Currently Playing, Total Members now reflect user's access
