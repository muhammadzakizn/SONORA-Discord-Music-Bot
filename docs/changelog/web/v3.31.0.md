# v3.31.0 - Notification Dialog UI Redesign

**Release Date:** 2025-12-30

## New Features

### Notification Dialog Redesign (macOS-style)

#### Ultra-Transparent Liquid Glass Effect
- 50% opacity background with 40px blur
- 180% saturation for enhanced glass effect
- Smoother backdrop (30% opacity)

#### Notification Cards
- Rounded rectangular cards (rounded-2xl)
- Gradient app icon (pink to purple) with shadow
- Unread indicator with pink ring glow
- Relative time display (now, 3m ago, 2h ago, 5d ago)

#### Notification Grouping by Type
- Notifications grouped by category: General, Changelog, Announcement, Personal
- Collapsed stack effect with visual stacked cards behind top card (h-20 with borders)
- Click to expand group, "Show less" button to collapse
- Badge showing "+N" for hidden notifications
- "Tap to show N more" hint on collapsed cards

#### Clear All Button Enhancement
- X icon by default
- Expands to show "Clear All" text on hover
- Smooth width transition animation

#### Delete Button
- Always visible on right edge of card (subtle on desktop)
- Hover to make more prominent with red color
- Easy one-tap delete

### Mobile Features
- **Swipe-to-Delete** - Swipe notification card right to delete
- **Grab Handle Bar** - Visual bar at bottom of panel
- **Swipe-Up to Close** - Swipe up on grab handle to close panel
- **Touch-Friendly** - Cards scale slightly during drag interaction
- **Swipe Hint** - "Swipe right to delete" text shown on mobile

### Action Buttons
- Mark as read (checkmark icon) - top right on hover
- Delete notification (X icon) - right edge, always accessible
- Visible on hover with smooth opacity transition

## Files Changed
- `components/NavLiquidGlass.tsx` - Notification dialog UI redesign for both Developer and Regular user dialogs
