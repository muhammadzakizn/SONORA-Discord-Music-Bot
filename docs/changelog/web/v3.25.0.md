# Web v3.25.0 - Shared Footer Component

**Tanggal**: 22 Desember 2025

## Perubahan UI

### Shared Footer Component
- Membuat komponen `Footer.tsx` bersama di `/components/`
- Footer sekarang konsisten di semua halaman:
  - Home page
  - Admin Dashboard (semua menu)
  - Developer Dashboard (semua menu)
  - Terms of Service
  - Privacy Policy
  - Support
  - Changelog
  - Documentation
  - Status page

### Perubahan File
- **Baru**: `components/Footer.tsx` - Komponen footer reusable
- **Diperbarui**: 
  - `app/page.tsx` - Menghapus inline Footer, gunakan shared component
  - `app/admin/layout.tsx` - Tambah Footer di main content
  - `app/developer/layout.tsx` - Tambah Footer di main content
  - `app/terms/page.tsx`, `privacy/page.tsx`, `support/page.tsx`
  - `app/changelog/page.tsx`, `docs/page.tsx`, `status/page.tsx`
