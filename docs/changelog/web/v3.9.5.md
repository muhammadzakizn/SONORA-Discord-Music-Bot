# Web Changelog v3.9.5 - Apple Music Lyrics Style

**Release Date:** 2025-12-18

## Lyrics Display
- Past lyrics now remain visible but dimmed (`opacity-40`) instead of collapsing.
- Future lyrics are grayed (`opacity-50`) without blur effect.
- Current line stays white and fully visible.

## Animation
- Background animation speed increased by 25% (time divisor reduced from 3500 to 2625).
