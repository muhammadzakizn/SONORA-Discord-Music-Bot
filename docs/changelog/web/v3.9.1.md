# Web Changelog v3.9.1 - Apple Music Lyrics UI

**Release Date:** 2025-12-18

## UI Improvements

### Lyrics Player Layout
- **50:50 Layout**: Equal width for album/controls panel and lyrics panel
- Removed `max-w-[500px]` constraint from left panel

### Apple Music-Style Dynamic Lyrics
- **Past lyrics**: Collapse to height 0 with smooth transition
- **Current lyric**: Highlighted with word-by-word glow animation
- **Future lyrics**: Visible but blurred (2px blur filter)

### Scroll Behavior
- Manual scrolling reveals all lyrics temporarily
- Auto-resumes karaoke mode after 3 seconds of no scroll
- Auto-scroll to current line when not manually scrolling

## Technical Changes
- Added `isUserScrolling` state for scroll detection
- Added `scrollTimeoutRef` for 3-second timeout
- Added `onScroll` handler on lyrics container
- CSS transitions for smooth collapse/expand (500ms duration)
