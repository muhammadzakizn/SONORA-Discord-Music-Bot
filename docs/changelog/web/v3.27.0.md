# Web v3.27.0 - Dynamic Time-Based Welcome Message

**Date**: December 23, 2025

## UI Changes

### TimeBasedGreeting Component
- **Dynamic Greetings**: Greeting message changes based on user's local time (morning, afternoon, evening, night, midnight)
- **Random Messages**: Pool of 5 different greeting messages for each time period
- **Animated Illustrations**: SVG-based illustrations with Framer Motion animations for each time period:
  - Morning: Rising sun with floating clouds
  - Afternoon: Bright sun with radiating rays
  - Evening: Sunset with flying birds
  - Night: Crescent moon with twinkling stars
  - Midnight: Full moon with shooting star

### Technical Details
- Uses `new Date()` for accurate local device time detection
- Updates time check every minute
- SSR-safe with hydration handling
- Smooth animations with Framer Motion

### File Changes
- **New**: `components/admin/TimeBasedGreeting.tsx` - Dynamic greeting component with SVG illustrations
- **Modified**: `app/admin/page.tsx` - Integrated TimeBasedGreeting replacing static welcome message
