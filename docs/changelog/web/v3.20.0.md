# Web Changelog v3.20.0 - Fullscreen Lyrics Improvements

**Release Date:** 2025-12-19

## Bug Fixes

### Lyrics Scroll Bug Fixed
- Fixed erratic scroll on track change
- Fixed jump to top on pause/resume
- Added `trackChangingRef` to prevent scroll during transitions

### Loading State
- Shows "Loading lyrics..." spinner while fetching
- Shows "Lyrics unavailable" if failed

### Track Transitions
- Smooth 600ms transition between tracks
- `previousTrack` state for animation reference

## Improvements

### Album Only Mode
- Removed placeholder text
- Album + controls properly centered when lyrics off

### Background Animation
- Slowed down WebGL rotation (5000 divisor vs 2625)
- Smoother, less distracting motion

## Files Modified
- `FullscreenLyricsPlayer.tsx` - Track transitions, loading states, scroll fix
- `WebGLBackground.tsx` - Slower animation speed
