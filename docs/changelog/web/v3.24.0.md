# SONORA Web v3.24.0 - Lyricify Lyrics Source

**Release Date:** 2025-12-22

## Changes

### Fullscreen Lyrics Player

#### New Lyrics Source
- **Lyricify (QQ)**: New source option for syllable-timed lyrics from QQ Music
- Positioned second in source list after Apple Music
- Displays as "Lyricify (QQ)" for clarity

#### Source Selection Updates
- Source order: Apple Music → Lyricify → Musixmatch → LRCLIB → Auto
- `lyricsSource` type extended to include `'lyricify'`

### Backend Integration
- Flask API supports `source=lyricify` parameter
- Lyricify lyrics cached server-side (10 min TTL)
- `available_sources` now includes lyricify

- `src/components/lyrics/FullscreenLyricsPlayer.tsx`
- `web/api/app.py`
