# Web Dashboard Changelog - v3.5.0

## 2024-12-13

### Added
- **Developer Dashboard Major Upgrade** - Complete redesign with Admin-style UI
- **Dashboard Overview** - New system stats and quick actions dashboard
- **Real-time Console** - Log filtering, search, export, and pause functionality
- **System Monitoring** - Live CPU, Memory, Disk metrics with history graphs
- **Bot Controls** - Shutdown, Restart, Pause/Resume All, Maintenance Mode
- **Server Management** - View and manage connected Discord servers
- **Broadcast Messaging** - Send announcements to servers with preview
- **User Management** - View and ban/unban users

### Backend API
- New `/api/developer/stats` endpoint for system metrics
- CPU, Memory, Disk usage monitoring via psutil
- Bot statistics: servers, voice connections, active users

### UI/UX Improvements
- Liquid Glass sidebar design matching Admin Dashboard
- Dark/Light mode support on all pages
- Responsive layout for mobile devices
- Smooth animations and transitions with Framer Motion
