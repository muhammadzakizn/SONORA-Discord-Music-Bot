# Web Dashboard Changelog - v3.6.0

## 2024-12-13

### Added
- **Maintenance Mode System** - Full maintenance management with progress tracking
- **Ban Management** - User and Server ban system with duration and reason
- **Admin Support Dashboard** - Ticket management from Admin panel

### Maintenance Features
- Activate/deactivate maintenance mode
- Progress percentage slider (0-100%)
- Progress stage selector (Starting, Backup, Updating, Testing, Finalizing, Complete)
- Live status message visible to users
- Maintenance history log
- Status Page integration

### Ban Features
- Ban users from using bot commands
- Ban servers from using bot
- Duration options: 1 day, 7 days, 30 days, 90 days, permanent
- Reason field (required)
- Support link included in ban message
- Unban/Reban functionality
- Search and filter by type or status

### Support Dashboard (Admin)
- View all support tickets
- Filter by status: Open, In Progress, Resolved, Closed
- Reply to tickets from dashboard
- Change ticket status
- Priority levels: Low, Medium, High, Urgent
- Ticket statistics overview
