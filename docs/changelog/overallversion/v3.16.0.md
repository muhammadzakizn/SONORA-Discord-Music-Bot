# SONORA v3.16.0 - YTDLP API Integration

**Release Date:** 2025-12-19

## New Features

### YTDLP API Integration
Bot uses YTDLP API for all audio operations with CLI fallback.

- `ytdlp_client.py` - API client class
- `youtube.py` - Updated to use API first
- Automatic fallback to CLI if API unavailable

## Technical Notes
- API runs on `localhost:9072`
- All methods have graceful fallback
