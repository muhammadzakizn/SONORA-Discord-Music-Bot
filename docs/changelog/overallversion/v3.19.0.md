# SONORA v3.19.0 - Synchronized Playback Dashboard

**Release Date:** 2025-12-19

## New Feature

Dashboard synchronized with Discord playback via WebSocket.

### API
- `GET /api/state/<guild_id>` - Track state
- `POST /api/state/<guild_id>/ack` - Dashboard ACK

### WebSocket
- `track_preparing`, `track_ready`, `playback_started`, `track_ended`
