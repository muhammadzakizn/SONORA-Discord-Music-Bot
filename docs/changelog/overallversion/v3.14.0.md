# SONORA v3.14.0 - YTDLP Audio API

**Release Date:** 2025-12-19

## New Features

### YTDLP Audio API
REST API terpisah untuk audio streaming via yt-dlp. Mencegah masalah cookies/streaming expired dengan menyediakan fresh stream URL on-demand.

**Endpoints:**
- `GET /api/ytdlp/search?q=<query>` - Search YouTube Music
- `GET /api/ytdlp/stream-url?title=<title>&artist=<artist>` - Fresh stream URL
- `GET /api/ytdlp/download?title=<title>&artist=<artist>` - Download audio
- `GET /api/ytdlp/stream?title=<title>&artist=<artist>` - Proxy stream
- `GET /api/ytdlp/info` - API status

## Technical Notes
- New Flask blueprint: `web/api/ytdlp_api.py`
- Auto-cleanup downloaded files after serving
- Fallback to download if stream URL fails
- Proxy streaming to avoid 403 errors
