# v3.9.9 - Apple Music Style Lyrics

**Release Date:** 2025-12-18

## Web Changes

### New Fullscreen Lyrics Player
- Apple Music-inspired synchronized lyrics display
- Per-word animated highlighting that follows playback
- Dynamic blur background from album artwork
- Vinyl disc spinning animation while playing

### Lyrics Player Features
- Play/Pause/Skip/Stop controls within lyrics view
- Collapsible queue panel to see upcoming tracks
- Romanization toggle for Asian lyrics (Japanese, Korean, Chinese)
- Progress bar with current time and duration

### Backend API
- New `/api/guild/<guild_id>/lyrics` endpoint
- Returns synced lyrics with word-level timing estimation
- Real-time playback position for smooth client-side sync

### UI Additions
- "Lyrics" button (Mic icon) in Guild Detail page playback controls
- CSS animations for vinyl spin effect and scrollbar hiding
