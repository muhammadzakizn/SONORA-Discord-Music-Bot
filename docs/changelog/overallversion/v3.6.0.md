# SONORA v3.6.0 - Maintenance, Bans & Support System

**Release Date:** 2024-12-13

## Summary
Implemented Phase 3-6 of the major upgrade: Maintenance Mode system with progress tracking, User/Server Ban Management, and Admin Support Dashboard.

---

## Developer Dashboard - Maintenance

### Maintenance Mode Page
- Activate/deactivate maintenance mode
- Progress percentage slider (0-100%)
- Progress stage selector with 6 stages
- Live status message for users
- Completion dialog with summary
- Maintenance history log
- Status Page integration

---

## Developer Dashboard - Ban Management

### Ban System
- User ban with reason and duration
- Server ban with auto-leave capability
- Duration options: 1 day, 7 days, 30 days, 90 days, permanent
- Required reason field
- Support link in ban messages (sonora.muhammadzakizn.com/support)
- Ban statistics: active, users, servers
- Unban/Reban functionality
- Search and filter by type or status

---

## Admin Dashboard - Support

### Support Dashboard
- View all support tickets
- Filter by status: Open, In Progress, Resolved, Closed
- Reply to tickets directly
- Change ticket status inline
- Priority levels: Low, Medium, High, Urgent
- Ticket statistics overview
- Ticket detail modal with message history
