# SONORA v3.4.0 - Dashboard Enhancement Phase 2

**Release Date:** 2024-12-12

## Summary
Enhanced web dashboard with Discord notifications, queue management, and UI improvements.

---

## Web Dashboard Changes
- Discord notifications for control actions (pause/resume/skip/stop)
- Queue management UI with move up/down and delete
- Smooth progress bar animation using requestAnimationFrame
- GIF image format support for server icons
- Image fallback for broken artwork

## Bot API Changes  
- Queue remove endpoint: `POST /api/queue/{guild_id}/remove/{position}`
- Queue move endpoint: `POST /api/queue/{guild_id}/move`
- Control actions send embed notifications to Discord channel
- Username tracking for action attribution

## PWA Changes
- Updated offline.html with blur backdrop
- Added Home and Status buttons to offline dialog
- Service worker cache version updated to v2
