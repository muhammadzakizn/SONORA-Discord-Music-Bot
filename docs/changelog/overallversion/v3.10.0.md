# SONORA v3.10.0 - Musixmatch Richsync Integration

**Release Date:** 2025-12-18

## Overview
Added Musixmatch as a lyrics source with richsync (syllable-level timing) support for enhanced Apple Music style lyrics display.

## New Features

### Musixmatch Lyrics Provider
- **Richsync Support**: Per-syllable timing for smooth karaoke-style animation
- **Auto Signature Auth**: Uses reverse-engineered web API with automatic signature generation
- **Fallback**: Falls back to regular lyrics if richsync unavailable

### Lyrics Source Selection
- **Admin Dashboard**: Choose between Auto, Musixmatch, or LRCLIB sources
- **Dynamic Switching**: Switch sources without closing the lyrics player
- **Current Source Display**: Shows which source is currently active

## Technical Changes

### Backend
- New `MusixmatchFetcher` class in `services/lyrics/musixmatch.py`
- Updated `LyricLine` model with `words` field for word-level timing
- Added `?source=` parameter to `/api/guild/<id>/lyrics` endpoint

### Frontend
- Lyrics source selector in settings dropdown
- Real-time source switching
- Current source badge display

## API Changes

```
GET /api/guild/{guild_id}/lyrics?source=musixmatch

Response includes:
- lyrics_source: Current source used
- available_sources: ["auto", "musixmatch", "lrclib"]
```
