# SONORA v3.11.5 - Apple Music Lyrics Style

**Release Date:** 2025-12-18

## Visual Improvements
- **Lyrics Visibility**: Past lyrics now stay visible but dimmed (40% opacity) instead of completely disappearing.
- **Future Lyrics**: Future lyrics are grayed out (50% opacity) without blur effect.
- **Current Line**: The active lyric line remains fully white and centered.
- **Background Animation**: Sped up WebGL background rotation by 25% for a more immersive feel.
