# SONORA v3.21.0 - Syllable Detection Lyrics

**Release Date:** 2025-12-19

## Overview
Smart lyrics highlighting that detects syllable timing availability.

## Changes
- **Syllable Detection**: Per-word glow only when real data exists
- **Line Highlight**: Full line glow for estimated timing (rap/fast lyrics)
- Cleaner display for songs without syllable data

## Technical
- `has_syllable_timing` flag in LyricsData model and API
