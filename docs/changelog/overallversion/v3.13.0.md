# SONORA v3.13.0 - Apple Music Lyrics Integration

**Release Date:** 2025-12-19

## New Features
- **Apple Music Lyrics**: Added Apple Music as primary lyrics source using media-user-token
- **TTML Format Support**: Parse Apple Music's TTML lyrics format with line timing
- **Word Timing Estimation**: Automatically estimate word timing for karaoke effect

## UI Changes
- Added "Apple Music" option to lyrics source selector (default)
- Source selector now shows: Apple Music, Musixmatch, LRCLIB, Auto
- Buttons wrap on smaller screens

## Technical Notes
- New `AppleMusicFetcher` service at `services/lyrics/applemusic.py`
- Reads `media-user-token` from `cookies/apple_music_cookies.txt`
- Falls back to other sources if Apple Music fails
