# v3.27.0 - Dynamic Time-Based Welcome Message

**Date**: December 23, 2025

## Summary
Admin Dashboard now features a personalized greeting system that adapts to the user's local time with animated illustrations.

## Web
- **TimeBasedGreeting**: New component with time-based greetings and SVG illustrations
- **5 Time Periods**: Morning, Afternoon, Evening, Night, Midnight
- **Animated SVGs**: Sunrise, sun, sunset, moon, and stars with smooth animations
- **Random Greetings**: Variety of messages for each time period
