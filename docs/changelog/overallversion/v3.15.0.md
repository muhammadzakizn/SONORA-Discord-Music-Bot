# SONORA v3.15.0 - One Track One Process Mode

**Release Date:** 2025-12-19

## New Features

### One Track One Process Mode (`ONE_TRACK_ONE_PROCESS`)
Sequential playlist processing mode yang lebih hemat resources.

- **Default:** `false` (buffer 3 tracks ahead)
- **When enabled:** `true` (buffer 1 track ahead)

Aktifkan di `.env`:
```env
ONE_TRACK_ONE_PROCESS=true
```

## Technical Notes
- Dynamic `buffer_size` property in `playlist_cache.py`
- Updated `_prefetch_next_track` limit in `media_player.py`
- Useful for limited resources or slow connections
