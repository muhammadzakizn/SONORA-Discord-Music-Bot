# v3.9.3 - Multi-Platform Audio Support

**Release Date:** 2025-12-16

## New Features

### TIDAL & SoundCloud Support
- Added TIDAL URL detection and processing (tracks, albums, playlists)
- Full SoundCloud support via native yt-dlp
- SoundCloud sets/playlists supported

### WhisperLRC Auto-Lyrics
- AI-generated synced lyrics from audio using Whisper
- Disabled by default (set `ENABLE_WHISPER_LYRICS=true` to enable)
- New lyrics chain: LRCLIB -> Syncedlyrics -> WhisperLRC -> Genius

### MusicDL Integration (Fallback)
- Created `services/audio/musicdl_handler.py` for multi-platform support
- Supports TIDAL, SoundCloud, and other platforms as fallback

## Files Modified
- `utils/validators.py` - Added TIDAL/SoundCloud URL patterns
- `services/audio/playlist_processor.py` - Added platform handlers
- `services/audio/musicdl_handler.py` - NEW: MusicDL wrapper
- `services/lyrics/whisper_lyrics.py` - NEW: WhisperLRC fetcher
- `services/metadata/processor.py` - WhisperLRC lyrics fallback
- `requirements.txt` - Added musicdl, faster-whisper

## Known Issues
- MusicDL requires `rich>=13.9.4` (SONORA has `rich==11.0.0`)
- TIDAL currently falls back to YouTube Music until rich is upgraded
