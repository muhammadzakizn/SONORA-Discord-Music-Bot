# Notification Sounds

## Current Sound
- `notification.ogg` - Unified notification sound for all types
  - Source: [Conclusive - notificationsounds.com](https://notificationsounds.com/notification-sounds/conclusive-message-tone)
  - Format: OGG Vorbis
  - Duration: ~1 second

## Customization
To change the notification sound:
1. Replace `notification.ogg` with your preferred sound
2. Keep the same filename and extension
3. Recommended: 1-2 seconds, moderate volume

## Format Requirements
- Format: OGG (current), MP3, or WAV
- Duration: 1-2 seconds recommended
- Volume will be played at 50%
